#  DualSenseX Steam

**DualSenseX Steam** is a powerful tool that enhances your PlayStation 5 DualSense controller experience on PC with Steam. It enables advanced features like adaptive triggers, haptic feedback, motion controls, and custom profiles.

## Key Features
- Full DualSense adaptive trigger support
- Haptic feedback and rumble enhancement
- Motion control (gyro) emulation
- Custom controller profiles per game
- LED color customization
- Battery level display
- Seamless Steam integration

## Getting Started
1. **Download** the latest version using the button above.
2. **Extract** the archive.
3. **Run** `DualSenseX.exe` as Administrator.
4. **Connect your DualSense controller** via USB or Bluetooth.
5. **Configure settings** and create profiles for your favorite games.

**Note:** For best results, use the controller in wired mode initially.
