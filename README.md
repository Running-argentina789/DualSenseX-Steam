## 🔧 Troubleshooting Tips

If you encounter issues, try these common fixes:

*   **Controller not detected:** Check your cable. Some USB cables only carry power and do not transfer data. Switch to a known high-quality data cable.
*   **Game ignores input:** Some games detect both the real controller and the virtual controller. If you experience double inputs, use the "Hide Controller" feature inside the app settings to mask the original device.
*   **App crashes on launch:** Ensure you have the latest .NET Desktop Runtime installed from the official Microsoft website.
*   **Bluetooth drops:** Bluetooth signals undergo interference. Keep your controller near your computer or use a wired connection for the most stable experience.
*   **Driver errors:** Reinstall the app. The installer automatically cleans up and repairs broken driver files during the process.
